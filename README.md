Texera Construções - Web App Oficial
Este é o site oficial da Texera Construções, desenvolvido com foco em conversão de clientes, automação de atendimento e alta performance. O projeto utiliza uma arquitetura SPA (Single Page Application) moderna, proporcionando uma navegação fluida sem recarregamentos de página.

🚀 Funcionalidades Principais
Chatbot Inteligente (FAQ + Orçamento): Um assistente virtual que resolve dúvidas frequentes (onde atendemos, formas de pagamento) e qualifica potenciais clientes através de um fluxo de 4 etapas antes de direcioná-los ao WhatsApp.

Design de Conversão (UX/UI): Chamadas para Ação (CTAs) estratégicas, como "Orçamento Grátis" e "Ver Obras", otimizadas para aumentar a taxa de cliques.

Galeria Interativa com Lightbox: Visualização detalhada dos serviços com sistema de zoom e descrições técnicas integradas.

Painel Administrativo Oculto: Sistema exclusivo para o proprietário monitorar o número de visitas, quantidade de orçamentos pedidos e atualizar o número do WhatsApp sem mexer no código.

Layout Responsivo: Totalmente adaptado para dispositivos móveis, tablets e computadores.

🛠️ Tecnologias Utilizadas
HTML5: Estrutura semântica.

CSS3: Estilização moderna com variáveis (CSS Variables), Glassmorphism e animações fluidas.

JavaScript (Vanilla): Lógica SPA, manipulação de DOM e persistência de dados via localStorage.

Font Awesome: Ícones vetoriais para melhor acessibilidade visual.

🔐 Acesso ao Painel Administrativo
Para acessar as métricas e configurações do site:

Na página inicial, clique 5 vezes seguidas na Logomarca da Texera.

Insira a senha padrão (definida inicialmente como 1234).

Lá você poderá visualizar o contador de visitas e atualizar o número do WhatsApp de destino.

📋 Como instalar e rodar
Certifique-se de que todas as imagens mencionadas acima estejam na pasta /img.

Mantenha os três arquivos (index.html, style.css, script.js) no mesmo diretório raiz.

Abra o arquivo index.html em qualquer navegador moderno ou hospede em serviços como GitHub Pages, Vercel ou Netlify.

✍️ Manutenção das Fotos
Para atualizar as fotos da galeria no futuro:

Substitua o arquivo na pasta img/ mantendo o mesmo nome.

Ou altere o atributo data-desc no index.html para atualizar a legenda que aparece no zoom da imagem.
