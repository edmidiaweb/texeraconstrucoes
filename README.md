# 🏗️ Texera Construções - Site Institucional

Este é um projeto de site institucional moderno e responsivo desenvolvido para a **Texera Construções**, empresa com mais de 30 anos de experiência no setor de construção civil em Guarujá e região.

O projeto conta com navegação multi-página, carrossel de serviços, formulário inteligente integrado ao WhatsApp e um chatbot personalizado para atendimento automático.

## 🚀 Funcionalidades

- **Home Interativa**: Design limpo com logo centralizada e menu de navegação horizontal estratégico no rodapé.
- **Página Quem Somos**: Texto focado em autoridade e tradição de mercado.
- **Galeria de Serviços**: Carrossel responsivo apresentando os principais serviços e faixas de preço.
- **Formulário de Pedido**: Coleta de dados com máscara de telefone automática e envio direto para o WhatsApp do proprietário.
- **Chatbot Inteligente (Widget)**: Assistente virtual que responde dúvidas comuns (preços, horários, contato) e direciona para atendimento humano.

## 🛠️ Tecnologias Utilizadas

- **HTML5**: Estrutura semântica das páginas.
- **CSS3**: Estilização personalizada, layout Flexbox e animações de interface.
- **JavaScript (Vanilla JS)**: 
  - Lógica de processamento de palavras-chave do Chatbot.
  - Injeção dinâmica de componentes (Widget de Chat).
  - Máscaras de input e integração com API do WhatsApp.
- **GitHub Pages**: Hospedagem gratuita e deploy contínuo.
